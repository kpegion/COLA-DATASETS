    comment :
    GFDL experiment name = c180_hiram_H3. CMIP5 experiment name = AMIP (r3i1p1). Model was run in AMIP mode with SST and sea ice concentration determined from the HadISST1 data set (http://hadobs.metoffice.com/hadisst/). The sea-ice model assumes that each grid point is either fully ice-covered or ice-free and also assumes a uniform ice thickness of 2 meters. This realization was started on 1976 January 1 from initial conditions taken at 1978 January 1 from the GFDL-HIRAM-C180 AMIP experiment (r1i1p1). The first three years of this integration (1976-1978) have not been published.
